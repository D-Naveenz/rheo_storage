# Rheo Storage

Rust-first rewrite of `Rheo.Storage`.

## Workspace
- `rheo_storage_lib`: core runtime library
- `rheo_storage_def_builder`: definitions package inspection and normalization tool
- `rheo_storage_com`: classic COM wrapper DLL for general C#/.NET desktop consumers
- `rheo_storage_winrt`: WinRT-facing wrapper crate for packaged Windows consumers
